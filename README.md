# Aurora.Shop – Nuxt 3 E‑Store (Portfolio Demo)

A modern e‑commerce demo built with **Nuxt 3**, designed specifically to look great in a portfolio.

## Tech Stack

- Nuxt 3 (SSR)
- TypeScript
- Pinia for state (cart, auth)
- SCSS for styling

## Getting Started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Features (planned)

- Modern landing page
- Product catalog and product details
- Cart with quantity and totals
- Checkout demo flow
- Fake auth + admin demo section

You can extend this to connect to a real backend or Stripe later if you want real payments.
